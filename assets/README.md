# HabitFlow Assets

This directory contains assets for the HabitFlow application.

## Subdirectories

- `icons/`: Contains application icons, launcher icons, and feature-specific icons.
- `images/branding/`: Contains branding assets such as logos, splash screen images, and marketing materials.

## Placeholders

Currently, this project uses standard Flutter placeholders. Production-quality assets (SVG/PNG) should be added here during the branding phase.
