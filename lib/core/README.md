# Core Layer

The `core/` directory contains the foundational building blocks of the application that are independent of any specific feature.

## Structure
- `constants/`: Global constants, strings, and keys.
- `errors/`: Custom exception classes and failure handling.
- `extensions/`: Dart extension methods to enhance existing types.
- `services/`: Low-level infrastructure services (e.g., local storage, network clients).
- `theme/`: App-wide styling, colors, and typography.
- `router/`: Navigation and routing configuration.
- `utils/`: Helper functions and utility classes.
