# Assets Layer (Code-bundled)

The `lib/assets/` directory (as per project structure requirements) contains references or bundled asset-related logic. 

*Note: Physical assets are typically stored in the root `assets/` directory, while this directory can be used for generated code (e.g., `assets.gen.dart`) or static asset paths defined in Dart.*

## Structure
- `images/`: Photographic images.
- `icons/`: Vector or bitmap icons.
- `illustrations/`: Custom artwork and drawings.
- `animations/`: Lottie or Rive animation files.
- `fonts/`: Custom typography files.
- `quotes/`: Static content or JSON files for quotes.
